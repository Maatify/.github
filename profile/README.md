# Maatify

A modular, architecture-first PHP ecosystem focused on:
- Deterministic design
- Zero magic
- Explicit configuration
- Strict boundaries
- Long-term maintainability

## Core Principles
- No hidden behavior
- No runtime guessing
- DTO-driven configuration
- DI-first architecture
- Testable by design

## Main Packages
- maatify/data-adapters
- maatify/infra-drivers
- maatify/security-guard
- maatify/rate-limiter
- maatify/common

## Development Model
- Phase-based execution
- Architecture lock before implementation
- Quad-AI workflow (Architect / Builder / Fixer / Executor)
